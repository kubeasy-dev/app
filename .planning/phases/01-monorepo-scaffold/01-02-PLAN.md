**3. Create `packages/api-schemas/src/challenges.ts`** covering all challenge router shapes:

Schemas to define (all field names match existing tRPC procedure shapes):

- `ChallengeDifficultySchema` = `z.enum(["easy", "medium", "hard"])`
- `ChallengeListInputSchema` = z.object with: difficulty (optional ChallengeDifficultySchema), type (optional string), theme (optional string), showCompleted (optional boolean default true), search (optional string)
- `ChallengeListItemSchema` = z.object with: id (number int), slug (string), title (string), description (string), theme (string), themeSlug (string), difficulty (ChallengeDifficultySchema), type (string), typeSlug (string), estimatedTime (number int), initialSituation (string), ofTheWeek (boolean), createdAt (z.coerce.date()), updatedAt (z.coerce.date()), completedCount (number int), userStatus (string nullable)
- `ChallengeListOutputSchema` = z.object with: challenges (array of ChallengeListItemSchema), count (number int)
- `ChallengeDetailSchema` = z.object with: id (number int), slug (string), title (string), description (string), theme (string), themeSlug (string), difficulty (ChallengeDifficultySchema), type (string), typeSlug (string), estimatedTime (number int), initialSituation (string), ofTheWeek (boolean), available (boolean), starterFriendly (boolean), createdAt (z.coerce.date()), updatedAt (z.coerce.date())
- `ChallengeGetBySlugOutputSchema` = z.object with: challenge (ChallengeDetailSchema nullable)
- `ChallengeObjectiveItemSchema` = z.object with: id (number int), objectiveKey (string), title (string), description (string), category (string), displayOrder (number int)
- `ChallengeGetObjectivesOutputSchema` = z.object with: objectives (array of ChallengeObjectiveItemSchema)
- `ChallengeCreateInputSchema` = z.object with: slug (string min 1), title (string min 1), description (string min 1), theme (string min 1), difficulty (ChallengeDifficultySchema), estimatedTime (number int positive), initialSituation (string min 1), ofTheWeek (boolean default false)
- `ChallengeCreateOutputSchema` = z.object with: success (boolean), challenge (z.any()), action (z.enum(["created", "updated"]))
- `ChallengeDeleteInputSchema` = z.object with: slug (string min 1)
- `ChallengeDeleteOutputSchema` = z.object with: success (boolean), message (string)
- `ChallengeSetAvailabilityInputSchema` = z.object with: slug (string), available (boolean)
- `SlugInputSchema` = z.object with: slug (string)

Export all schemas AND their inferred types (e.g., `export type ChallengeListInput = z.infer<typeof ChallengeListInputSchema>`)

CRITICAL: Do NOT import from `@/server/db/schema` or any `@/` path. Inline ALL enum values directly.

- `ChallengeSubmitInputSchema` = z.object with: challengeSlug (string), results (array of ObjectiveResultSchema min 1)
- `ChallengeSubmitSuccessOutputSchema` = z.object with: success (z.literal(true)), xpAwarded (number), totalXp (number), rank (string nullable), rankUp (boolean), firstChallenge (boolean), streakBonus (number), currentStreak (number)

- `ChallengeSubmitOutputSchema` = z.union of SuccessOutput and FailureOutput
